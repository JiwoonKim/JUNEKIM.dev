# baby dragon devlog
공부한 내용을 정리하는 블로그

## Developed With
- ReactJS
- GatsbyJS

## 블로그 포스트 작성
- `content` 폴더 내에 마크다운(.md) 파일 작성
- (폴더 새로 생성 후 파일 작성도 가능)
-  다음과 같은 내용 반드시 포함

```markdown
---
path: "/패스지정" // 파일이 어떤 폴더에 있든 이 패스만 있으면 알아서 
date: '2018-01-14' // 날짜
title: 제목
description: 짧은 설명
image: ''
tags: ['태그1','태그2','태그3'] // 원하는 태그 달아주기
---

여기에 내용 작성

```
